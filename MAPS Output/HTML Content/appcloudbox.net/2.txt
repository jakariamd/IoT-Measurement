<div><h1>  <b> Privacy Policy </b></h1><p>  Air Launcher (“Air Launcher” or “we") collects information from you in order to provide better service and a better user experience. What information we use and how we use it: </b></h1><p>  1. Installed apps information</p><p>  The only purpose we upload the installed apps information is to access apps classification labels so that we can help you organize the installed apps into folders by categories automatically. We don't do it otherwise.</p><p>  2. Take pictures and videos</p><p>  Flashlight is a useful feature in Air Launcher that uses your device’s camera flashlight and requires camera permission. Air Launcher will not take photos and videos by itself.</p></div> <p><p>  3. Find accounts on the device</p></p> <p><p>  Air Launcher does not recognize or save users' account information for any programs. We use this permission to read  email when you send feedback to us so that we can contact you when problem solved.</p></p> <p><p>  4. Precise location (GPS and network-based) and Approximate location (network-based)</p></p> <p><p>  This permission allows Air Launcher to provide local weather information for users. We will never upload any users’ location information.</p></p> <p><p>  5. Read your contacts</p></p> <p><p>  Air Launcher does not recognize or save users' contacts information for any programs. The only purpose using this permission is to help users search local contacts with search bar. We will never upload any users’ contacts information.</p></p> <p><p>  6. Read phone status and identity</p></p> <p><p>  The only purpose using this permission is to identify different users. We don't do it otherwise.</p></p> <p><p>  7. Read the contents of your USB storage</p></p> <p><p>  This permission allows users to view wallpapers stored on their SD card.</p></p> <p><p>  8. Modify or delete the contents of your USB storage</p></p> <p><p>  This permission allows users to delete  wallpapers stored on their SD card.</p></p> <p><p>  9. Set wallpaper and adjust your wallpaper size</p></p> <p><p>  This permission allows users to set a selected image as their wallpaper.</p></p> <p><p>  10. Control vibration</p></p> <p><p>  This permission allows Air Launcher to make the phone vibrate once after users set the device on vibrate, confirming the setting has been turned on.</p></p> <p><p>  11. Run at startup</p></p> <p><p>  Air Launcher will run as startup to provide users with a launcher service.</p></p> <p><p>  12. Toggle sync on and off</p></p> <p><p>  This permission allows Air Launcher to synchronize with the user's sync settings.</p></p> <p><p>  13. Read sync settings</p></p> <p><p>  This permission allows Air Launcher to recognize whether sync settings are turned on or off.</p></p> <p><p>  14. Read Home settings and shortcuts</p></p> <p><p>  This permission is used to acquire the user’s Home settings and shortcuts, in order to maintain user preferences.</p></p> <p><p>  15. Retrieve running apps</p></p> <p><p>  This permission is used in Boost in order to view the running apps.</p></p> <p><p>  16. Install shortcuts</p></p> <p><p>  This permission is used to install shortcuts on other launchers, so users can continue to use Air Launcher related functions.</p></p> <p><p>  17. Pair with Bluetooth devices</p></p> <p><p>  This permission is used in Air Launcher settings, in order to pair with Bluetooth devices.</p></p> <p><p>  18. Access Bluetooth settings</p></p> <p><p>  This permission is used in Air Launcher settings, in order to turn on and off the Bluetooth.</p></p> <p><p>  19. Connect and disconnect from Wi-Fi</p></p> <p><p>  This permission is used in Air Launcher settings in order to connect and disconnect from Wi-Fi.</p></p> <p><p>  20. Full network access andReceive data from Internet</p></p> <p><p>  This permission is used to access the device’s network for certain functions including downloading wallpapers and accessing app classification labels.</p></p> <p><p>  21. Change network connectivity</p></p> <p><p>  This permission is used in Air Launcher settings and notification toolbar, in order to change network connectivity.</p></p> <p><p>  22. View network connections</p></p> <p><p>  This permission is used in Air Launcher settings in order to view network connections.</p></p> <p><p>  23. View Wi-Fi connections</p></p> <p><p>  This permission is used in Air Launcher settings in order to view Wi-Fi connections.</p></p> <p><p>  24. Control flashligt</p></p> <p><p>  This permission is used to turn ON/OFF your device’s camera flashlight.</p></p> <p><p>  25. Prevent phone from sleeping</p></p> <p><p>  The only purpose using this permission is to prevent the phone from going to sleep when you are playing videos in fullscreen mode, we don't do it otherwise.</p></p> <p><p>  26. Apply for notification access</p></p> <p><p>  This access is used to display the number of unread messages in the form of badge on icon. We will never read the content of your messages or upload any users’ message information.</p></p> <p><p>  27. Accessibility</p></p> <p><p>  This permission is used to close apps chosen by users with accessibility.
