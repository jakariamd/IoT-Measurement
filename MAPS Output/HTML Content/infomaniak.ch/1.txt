Localisation will make it possible to retrieve the WiFi (SSID) to which the device is connected and to compare it to the list of authorized access points (SSID). If the network used is in the list, the application will synchronize the agenda(s) and / or address book(s) on the device. </li></ul></div> <div><h3>  How are your personal data shared?</h3><p>  Synchronization of the agendas and / or address books means that the events and contacts will be available for consultation on the end device / end application and are thus subject to the conditions of use of the device and the native or third-party applications used.</p></div> <div><h2>  Infomaniak Check (kCheck)</h2><p>  Infomaniak Check is an application which enables the user to send the elements requested in complete security in order to perform security controls. When the user receives an identity verification request (please consult the specific conditions to understand the different reasons), they are asked either to download the application or to use it if it is already installed on the device. All personal data communicated within the framework of an identity verification request will be stored while the request is processed and at most for 72 hours if the request is not finalised by the user, and will be deleted from our servers 24 hours after processing. Once the procedure has been finalised, the user can delete the application.</p></div> <div><h3>  What personal data does Infomaniak collect about me and for what purpose?</h3><p>  The purpose of the personal data collected by the application is to check the identity of the person following the procedure. Depending on the nature of the procedure, the following information may be collected:</p><ul>  <li>Mobile phone number: requested during the procedure</li><li>Geolocalisation: requested during the procedure via the device’s GPS</li><li>Identity documents: passport, identity card or driving licence (depending on the legislation in force in the user’s country) are photographed via the application</li><li>Selfie: the user is asked to take a selfie while holding the identity document close to the face using the application</li></ul></div> <div><h3>  How are your personal data used?</h3><p>  Your personal data are used as follows:</p><ul>  <li>Mobile phone number: used to send a verification code by SMS and the dialling code is used for cross-referencing the data with the geolocalisation data</li><li>Geolocalisation: used for cross-referencing the data with the dialling code of the phone number</li><li>Identity documents: used to check the identity of the user</li><li>Selfie: used to identify the user</li></ul></div> <div><h3>  How are your personal data shared?</h3><p>  These data are never shared with third-parties and are only processed internally.</p></div> <div><h2>  Gmail email migration via OAuth</h2><p>  Infomaniak offers the possibility of importing emails from other providers.
