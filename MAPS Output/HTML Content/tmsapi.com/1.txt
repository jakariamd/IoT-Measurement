It is specific to a program’s title, description language, and version. The tmsId is the industry’s gold standard for recognizing and synchronizing entertainment assets.The first two letters of the tmsId indicate entity type:               
</li><li><strong> SH </strong> = <strong> Show </strong> - includes TV specials, one-time shows, and series (non-episode-specific information)</li></ul></div> <ul><ul>  <li><strong> EP  </strong>=<em> Episode </em> - TV show episodes; metadata includes season number, episode number, and episode title; relates back to parent using a seriesId</li><li><strong> MV  </strong>=<em> Movie </em> - includes both theatrical releases and made-for-television films
</li><li><strong> rootId –  </strong>This numeric ID is used to identify all related program records with different title, description language, and version. </li><li><strong> seriesId </strong><strong> –  </strong>This numeric ID is equivalent to the rootId of series' parent program record.  Only TV series programs (SH header record and EP episode records) will include seriesId.</li></ul></ul> <p><p>  <strong> Sample IDs for programs of each type: </strong>(Note that this is a small sample of program type information.  For more detailed information, see <a href="/page/Program_Metadata" target="_blank"> Program Metadata </a>.)</p></p> <div><td>  seriesId</td><td>  rootId</td><td>  tmsId</td><td>  title</td><td>  episodeTitle</td><td>  eventTitle</td><td>  descLang</td><td>  entityType</td><td>  185044</td><td>  185044</td><td>  SH006883590000</td><td>  House</td><td>  ---</td><td>  ---</td><td>  en</td><td>  Show</td><td>  185044</td><td>  3056194</td><td>  EP006883590001</td><td>  House</td><td>  Pilot</td><td>  ---</td><td>  en</td><td>  Episode</td><td>  185044</td><td>  3056196</td><td>  EP006883590003</td><td>  House</td><td>  Paternity</td></div> <div><td>  ---</td><td>  en</td><td>  Episode</td><td>  ---</td><td>  3542039</td><td>  MV002523450000</td><td>  Avatar</td><td>  ---</td><td>  ---</td><td>  en</td><td>  Movie</td><td>  ---</td><td>  3542039</td><td>  MV003268290000</td><td>  Avatar: An IMAXExperience</td><td>  ---</td><td>  ---</td><td>  en</td><td>  Movie</td><p>  version 9 </p><span>  as of  </span><span>  by </span><span>  Raimond</span></div> <div><h2>  Docs Navigation</h2><p>  </p></div>
