</p></p> <p><p>  <strong> 2. How we use Cookie and similar technology </strong>
</p></p> <p><p>  <strong> 2.1 Cookie </strong>
</p></p> <p><p>  To ensure the proper functioning of our website, we sometimes store small data files called cookie on mobile devices. Cookie is a plain text file stored on a computer or mobile device by a web server. The content of Cookie can only be retrieved or read by the server that creates it. Each Cookie is unique to your mobile application. Cookie usually contains identifiers, site names, and some numbers and characters. We can store user preference data with Cookie.
</p></p> <p><p>  Most Internet service providers enable Cookie to improve user experience. With Cookie, it is possible to remember the user's single visit (using session Cookie) or multiple visits (using permanent Cookie). With Cookie, settings such as the language of the mobile device, personal data and other data can be saved. This means that users do not need to reconfigure user preferences every time they visit. If a mobile device does not use Cookie, every time a user turns on the mobile device, the mobile device will treat it as a new visitor. For example, if you have logged in to a mobile device, the device will not recognize you when you log in again, and you will be logged out again.
</p></p> <p><p>  We will not use Cookie for any purpose other than those described in this policy.
